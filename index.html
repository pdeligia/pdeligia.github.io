Pantazis Deligiannis

I am a senior research software engineer at Microsoft Research in Redmond, WA.
I enjoy working at the intersection of programming languages and (distributed) systems, and I am currently focused on researching and developing new programming technologies to help developers build highly-reliable and fault-tolerant cloud services.

I co-created P#, an asynchronous event-driven programming language and framework for designing, implementing and automatically testing distributed systems and microservices. P# is used in production by several Microsoft Azure teams. We recently renamed P# to Coyote, check out our new project homepage!

Prior to working at Microsoft, I studied for my PhD at Imperial College London, where I was fortunate to be advised by Alastair F. Donaldson. My PhD focused on scalable techniques for automatically analyzing and testing real-world asynchronous software systems. During my PhD, I did three research internships at Microsoft Research.

Projects.

I am currently working on:

  • [git] Coyote, a programming framework that simplifies designing, developing and testing asynchronous software. Coyote is the evolution of the P# project.
  • [git] P#, a programming language (implemented on top of C#) that simplifies designing, developing and testing asynchronous software.
  • Reliable State Machines (RSMs), an extension of P# that enables automatic fault-tolerance for cloud services.

Some cool projects I have previously worked on (list not up-to-date/complete):

  • Snowflake, an experimental extension of the .NET runtime that provides APIs for safe manual memory management alongside the traditional garbage collector.
  • Dungeon Story, a Puzzle/RPG game that I developed (as a hobby project) during summer 2012. The game is available worldwide in the iOS App Store, it has been downloaded by more than 195,000 users, and has hundreds of 4.5/5 star ratings.
